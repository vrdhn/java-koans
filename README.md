# java koans


Requires java 21:
```
$ java --version
openjdk 21.0.1 2023-10-17 LTS
...
```

To run:
```
java Run.java
```

Use any editor or IDE to edit the koans in `koans/*Koan.java` files


# Rules for writing koans:

```
public class KoanSomething {
     public static void koanSomething() {
         ...
     }
 }
 ```
